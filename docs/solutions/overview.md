# Solutions Overview

The Azure Local Toolkit includes placeholder folders for future solution packages. Active solutions have their own dedicated repositories.

## Active Solutions (Separate Repos)

| Solution | Repository | Description |
|----------|------------|-------------|
| **AVD** | [aurelocal-avd](https://github.com/AzureLocal/aurelocal-avd) | Azure Virtual Desktop on Azure Local |
| **SOFS** | [azurelocal-sofs-fslogix](https://github.com/AzureLocal/azurelocal-sofs-fslogix) | Scale-Out File Server for FSLogix profile containers |

## Future Solutions (Placeholders)

These folders exist in `solutions/` for future development and brainstorming:

| Solution | Description |
|----------|-------------|
| **AI Foundry** | AI/ML workloads on Azure Local |
| **AKS** | Azure Kubernetes Service on Azure Local |
| **App Services** | Azure App Services on Azure Local |
| **IoT Operations** | IoT workloads on Azure Local |
| **Machine Learning** | ML training and inference on Azure Local |
| **MI SQL** | Managed Instance SQL on Azure Local |

Contributions and ideas for these solutions are welcome — open an issue to start a discussion.
